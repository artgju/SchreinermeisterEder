# Schreinerei Meister Eder – Website

## Start

- Öffne `index.html` direkt im Browser, oder
- starte einen lokalen Server (empfohlen, z. B. VS Code Live Server Extension).

## Inhalte anpassen (wichtig)

In `index.html` sind aktuell **Platzhalter** hinterlegt:

- Telefon: `+49 000 0000000`
- E-Mail: `kontakt@schreinerei-meister-eder.de`
- Adresse: `Musterstraße 1, 00000 Musterstadt`
- Öffnungszeiten
- Kundenstimmen
- `schema.org` JSON-LD (Telefon/E-Mail/Adresse/URL)

Suche nach `Muster` bzw. `00000` und ersetze alles mit echten Daten.

## Dateien

- `index.html` – Inhalt/Struktur, CTAs, SEO/Schema.org
- `styles.css` – Premium-Layout, responsive, ohne externe Abhängigkeiten
- `script.js` – Menü (mobil), Header-Scroll-State, Footer-Jahr
